# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze and visualize sales performance data. It provides insights into sales trends, profit analysis, order distribution, and regional performance to support better business decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Track profit and order trends
- Compare sales across regions and categories
- Identify top-performing sub-categories
- Monitor business KPIs against targets

---

## 🛠 Tools & Technologies Used
- **Power BI**
- **DAX (Data Analysis Expressions)**
- **Data Visualization**
- **Data Cleaning & Transformation**

---

## 📈 Dashboard Features

### 🔹 KPI Cards
- Total Sales
- Total Profit
- Total Orders

### 🔹 Sales Trend Analysis
- Year-wise and quarter-wise sales analysis

### 🔹 Regional Analysis
- Sales distribution across different regions

### 🔹 Category Performance
- Comparison of Furniture, Office Supplies, and Technology

### 🔹 Sub-Category Analysis
- Best-performing product sub-categories

### 🔹 Shipping Analysis
- Order distribution by shipping mode

### 🔹 Geographic Visualization
- State-wise sales mapping

### 🔹 KPI Performance Tracking
- Actual profit vs target profit

---

## 📊 Key Insights
- Identified top-performing regions
- Analyzed category-wise contribution to total sales
- Tracked yearly sales growth
- Evaluated profit performance against business targets
